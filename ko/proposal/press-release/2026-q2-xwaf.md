# 📰 보도자료

## 큐비트시큐리티, OCI 기반 차세대 웹 보안 서비스 ‘xWAF.io’ 출시

### DDoS·크리덴셜 스터핑·제로데이 공격 대응… PLURA-XDR 연동으로 통합 보안 구현

큐비트시큐리티(Qubit Security Inc.)는 클라우드 인프라 전문 기업 오라클 클라우드 인프라스트럭처(OCI)를 기반으로 한 차세대 웹 보안 서비스 ‘**xWAF.io**’를 출시했다고 밝혔다.

xWAF.io는 최근 고도화되는 사이버 공격 환경에 대응하기 위해 설계된 **AI 기반 통합 웹 방화벽(Web Application Firewall)** 서비스로,
대규모 **DDoS 공격 대응**은 물론, 국내에서 반복적으로 발생하고 있는 **크리덴셜 스터핑 공격**, 그리고 알려지지 않은 취약점을 악용하는 **제로데이 공격**까지 효과적으로 방어할 수 있도록 개발됐다.

특히 크리덴셜 스터핑 공격은 탈취된 계정 정보를 활용해 대량 로그인 시도를 수행하는 공격으로, 국내 주요 서비스에서 지속적으로 발생하는 대표적인 위협이다. xWAF.io는 로그인 시도 패턴, 세션 행동, 요청 흐름을 종합적으로 분석하여 비정상 접근을 실시간으로 탐지하고 차단한다.

또한 기존 보안 솔루션이 탐지하기 어려웠던 제로데이 공격에 대해서도 **웹 요청 본문(Post-body) 기반 정밀 분석 기술**과 AI 기반 이상 탐지 기능을 결합하여, 사전 정의되지 않은 공격까지 탐지·차단할 수 있도록 했다. 이를 통해 단순 시그니처 기반 방어를 넘어서는 **행위 기반 보안 체계**를 구현했다.

특히 xWAF.io는 **AI와 연동하여 제로데이 공격을 탐지·차단하는 기능을 세계 최초로 구현**한 것이 특징이다. 기존 보안 체계가 알려진 공격 패턴에 의존하는 한계를 넘어서, 웹 요청 본문 데이터와 행위 패턴을 AI가 실시간으로 분석하여 알려지지 않은 공격까지 선제적으로 식별하고 대응할 수 있도록 설계되었다.

xWAF.io는 단독 서비스에 그치지 않고, 큐비트시큐리티의 정보보안 통합 플랫폼 **PLURA-XDR**과 연동되어 동작한다.
이를 통해 웹 공격 이벤트는 물론, 엔드포인트(EDR), 시스템 로그, 웹 요청 및 응답 로그 등 다양한 보안 데이터를 통합 분석할 수 있으며, 공격의 전후 맥락을 포함한 **확장된 탐지 및 대응(XDR)** 체계를 제공한다. 특히 웹 요청 본문 로그까지 포함한 심층 분석을 통해 기존 보안 체계에서 놓치기 쉬운 공격 흐름까지 식별할 수 있는 것이 특징이다.

큐비트시큐리티 신승민 대표는
“보안의 핵심은 공격을 얼마나 깊이 기록하고 분석할 수 있는가에 달려 있다”며,
“xWAF.io는 OCI의 안정적인 글로벌 인프라 위에서 동작하며, AI 기반 제로데이 대응 기술과 PLURA-XDR 연계를 통해 차세대 통합 보안 플랫폼으로 확장될 것”이라고 밝혔다.

xWAF.io는 OCI 환경에 최적화된 구조로 설계되어 클라우드 환경에서 빠르게 적용 가능하며, 향후 Oracle Cloud 고객을 대상으로 한 보안 서비스 확장과 함께 글로벌 시장을 겨냥한 **클라우드 기반 보안 SaaS 모델**로 본격적인 사업을 전개할 계획이다.
