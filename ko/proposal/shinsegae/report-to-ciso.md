# 신세계 I&C PoC 악성코드 탐지 보완 결과 보고

**보고 목적:** 신세계 I&C PoC 악성코드 탐지 테스트 결과 및 보완 완료 현황 보고  
**대상:** 경영진 보고용  
**작성:** 큐비트시큐리티 / PLURA

---

## 1. PoC 일정

본 PoC는 신세계 I&C 환경에서 악성코드, 파일리스 공격, Exploit 샘플을 대상으로 PLURA-XDR의 행위 기반 탐지 가능성을 검증하기 위해 진행하였다.

| 구분     | 내용                                                   |
| ------ | ---------------------------------------------------- |
| PoC 기간 | 2025년 6월 26일 ~ 6월 27일                                |
| 테스트 대상 | 악성코드 / 파일리스 / Exploit 샘플 총 36종                       |
| 검증 범위  | 실행 행위, 프로세스 행위, 레지스트리 변경, C2 통신, 권한 상승, 파일 생성 및 삭제 등 |
| 점검 항목  | 악성코드 탐지, 기능 체크리스트, 원격 사고대응, 고객사별 로그 접근 통제            |

---

## 2. 진행 내용

PoC에서는 총 36종의 샘플을 대상으로 탐지 테스트를 수행하였다.

초기 테스트 결과, 일부 샘플은 정상 탐지되었으며, 미탐으로 분류된 8종에 대해서는 상세 행위 분석을 진행하였다.

분석 결과, 미동작 또는 악성 행위 로그가 발생하지 않은 3종은 보완 대상에서 제외하였고, 실제 행위가 확인된 5종에 대해서는 추가 탐지 정책 제작을 완료하였다.

추가 탐지 정책은 아래 행위를 중심으로 보강하였다.

| 보완 영역      | 주요 탐지 행위                            |
| ---------- | ----------------------------------- |
| 프로세스 행위    | 비정상 자식 프로세스 실행, 프로세스 인젝션, API 직접 호출 |
| 시스템 변경     | 레지스트리 값 변경, DLL 생성, 보안 정책 접근        |
| 네트워크 행위    | C2 통신 연결, 외부 연결 시도                  |
| 랜섬웨어 유사 행위 | 랜섬노트 생성, 대량 파일 생성, 권한 상승 시도         |
| 정보 탈취 행위   | 브라우저 파일 덤프, 흔적 삭제, 외부 통신            |

기능 체크리스트는 별도 항목으로 모두 확인 완료하였다.

그룹 관리, 고객사별 로그 접근 통제, Kubernetes 및 서버리스 환경 대응, 에이전트 리소스 안정성, 네트워크 격리 후 원격 사고대응 절차 등 운영 기능도 함께 검증하였다.

---

## 3. 결과

PoC 이후 보완 결과, 기존 미탐 8종 중 미동작 3종을 제외한 5종에 대해 탐지 정책 추가 제작을 완료하였다.

| 항목              |    결과 |
| --------------- | ----: |
| 전체 테스트 샘플       |   36종 |
| 초기 미탐 항목        |    8종 |
| 미동작 / 행위 미발생 제외 |    3종 |
| 추가 탐지 정책 제작 완료  |    5종 |
| 기능 체크리스트        | 전체 완료 |

이번 PoC를 통해 PLURA-XDR은 단순 파일 시그니처 기반 탐지가 아니라, 실제 악성 행위가 발생했을 때 Sysmon, Security Event, 네트워크 행위 로그를 기반으로 탐지·분석이 가능함을 확인하였다.

특히 Windows Defender에서 탐지하지 못하거나 일부만 차단한 샘플에 대해서도, PLURA-XDR의 행위 기반 탐지 정책을 통해 추가 대응이 가능하다는 점을 검증하였다.

---

## 4. 종합 의견

본 PoC를 통해 PLURA-XDR의 악성 행위 기반 탐지 체계를 검증하였다.

현재 기준으로 실제 행위가 확인된 5종에 대한 추가 탐지 정책 제작은 완료되었으며, 기능 체크리스트도 모두 완료되었다.

이를 통해 악성코드 실행 이후 발생하는 프로세스, 레지스트리, 네트워크, 권한 상승, 정보 탈취 행위에 대한 탐지·분석·대응 가능성을 확인하였다.

**따라서 본 PoC는 PLURA-XDR의 고객사 운영 환경 적용 가능성을 확인한 검증 사례로 판단한다.**
