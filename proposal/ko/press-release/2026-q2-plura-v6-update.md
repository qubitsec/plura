## 보도자료

## 큐비트시큐리티, 차세대 통합 보안 플랫폼 **PLURA-V6** 업데이트 발표

### Storm-1175·Mythos 등 고도화되는 최신 위협 대응 강화… 웹·시스템·엔드포인트 전 영역 통합 탐지 및 포렌식 분석 지원

큐비트시큐리티(Qubit Security Inc.)는 자사의 차세대 통합 보안 플랫폼 **PLURA-V6** 업데이트를 완료했다고 밝혔다. PLURA-V6는 클라우드 SaaS 기반으로 웹, 서버, 엔드포인트 영역의 로그와 행위를 통합 분석해 탐지, 대응, 포렌식까지 연결하는 XDR 플랫폼으로, 이번 업데이트를 통해 최신 위협에 대한 가시성과 실시간 대응 역량을 한층 강화했다.

최근 보안 업계에서는 웹에 노출된 자산을 발판으로 침투한 뒤, 내부 확산과 데이터 탈취, 랜섬웨어 실행으로 이어지는 고속·다단계 공격이 잇따라 확인되고 있다. Microsoft는 최근 **Storm-1175**가 웹 노출 시스템의 취약점을 악용해 초기 침투 후 데이터 유출과 랜섬웨어 배포까지 매우 빠르게 전개하는 사례를 공개했으며, **Mythos(미토스)** 와 같은 AI 기반 고도화 위협도 새로운 변수로 주목받고 있다.

이러한 공격은 **웹 취약점 악용 → 내부 확산 → 데이터 탈취 → 랜섬웨어 실행**으로 이어지는 복합형 흐름을 짧은 시간 안에 전개한다는 점에서 위협적이다. 특히 웹 서버, 리눅스 시스템, 엔드포인트를 넘나드는 공격에서는 개별 보안 장비만으로 전체 침해 흐름을 파악하기 어렵기 때문에, 실시간 탐지와 사후 포렌식을 함께 지원하는 통합 대응 체계가 더욱 중요해지고 있다.

큐비트시큐리티는 이러한 환경 변화에 대응하기 위해 **PLURA-V6**의 통합 분석 및 대응 기능을 강화했다. PLURA-V6는 웹 공격 로그와 시스템 이벤트, 엔드포인트 행위를 연계 분석해 공격 전후 맥락을 보다 입체적으로 식별할 수 있도록 설계됐으며, 알려진 공격뿐 아니라 정상 도구 악용, 비정상 행위 연계, 침해 이후 확산 징후까지 폭넓게 추적할 수 있도록 고도화됐다.

특히 이번 업데이트에서는 외부 노출 자산을 노리는 최신 공격과 AI 기반 신위협에 대응할 수 있도록, 웹 요청 정보와 서버·OS 로그, 엔드포인트 이벤트를 하나의 흐름으로 연결해 분석하는 역량을 강화했다. 이를 통해 침투 이전의 이상 징후부터 권한 확대, 내부 이동, 데이터 유출, 랜섬웨어 실행 단계까지 더 빠르게 탐지하고 대응할 수 있도록 지원한다.

또한 PLURA-V6는 침해사고 발생 이후에도 단순 경고 제공에 그치지 않고, **포렌식 관점의 증적 확보와 원인 분석**까지 함께 지원하도록 개선됐다. 이를 통해 기업은 무엇이 탐지되었는지뿐 아니라, 어떤 경로로 침투했고 내부에서 어떤 행위가 수행되었으며 어디까지 영향을 받았는지를 보다 신속하게 확인할 수 있다.

큐비트시큐리티의 **전윤상 수석은**  “최근 공격은 속도와 은폐, 다단계 연계 측면에서 과거와 완전히 다른 양상을 보이고 있다”며, “이제는 웹 따로, 서버 따로, PC 따로 보는 방식으로는 실제 침해 흐름을 이해하기 어렵다. PLURA-V6는 기록과 분석, 탐지와 대응, 그리고 포렌식까지 하나로 연결해 기업이 실제 해킹 공격에 더 빠르게 대응할 수 있도록 지원할 것”이라고 말했다.

큐비트시큐리티는 앞으로도 PLURA-V6를 중심으로 웹 공격, 제로데이 악용, 랜섬웨어, 정상 도구 악용 공격은 물론, **AI 기반 신위협**에 대한 대응 역량도 지속적으로 강화해 나갈 계획이다.
