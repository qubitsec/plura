### 핵심 구성 요소 (Core Components)

| 구분 | 주요 기능 | 핵심 역할 |
| --- | --- | --- |
| **WAF** | **웹 방화벽** | 웹 공격 차단 및 서비스 보호 |
| **EDR** | **단말 대응** | 실시간 위협 탐지 및 즉각 대응 |
| **SIEM** | **로그 통합** | 보안 로그 분석 및 통합 가시성 |
| **SOAR** | **운영 자동화** | 보안 프로세스 및 대응 자동화 |
| **Forensic** | **사후 조사** | 침해 사고 분석 및 증거 확보 |
| **VAS** | **취약점 관리** | 잠재적 보안 약점 진단 및 관리 |
