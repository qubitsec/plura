# 필상 시스템 — Day 90 보안/운영 문구·템플릿 카탈로그(i18n) · 상태/공지/티켓/고객안내

1. **날짜**

* 2024-12-16

2. **목표**

* **상태페이지/장애 공지/완화 안내/보안 경고/고객지원 안내/티켓 응답**에 쓰이는 **표준 문구·템플릿**을
  **ko/ja/en i18n**으로 일괄 관리하고 **톤·법적 고지·시간대**를 일치시켜 **혼선/지연 0건** 달성

---

3. **내용**

### A. 카탈로그 구조

```
/templates
  /status    # 상태페이지/작업 공지
  /incident  # 장애/완화/복구/후속 RCA 안내
  /security  # 보안 경고/권고/PSIRT 알림
  /support   # 고객지원/가이드/FAQ/요청 접수 회신
  common.yml # 공통 변숫값/서명/연락처/타임존
```

* **버전**: `tmpl-YYYY.MM.minor`로 태깅, 릴리스 노트/변경 diff 기록
* **공통 변수**: `{region}`, `{tenant}`, `{window}`, `{impact}`, `{next_update_at}`, `{signed_url}`, `{contact}` 등

### B. 톤 & 법적 고지(원칙)

* **사실/지표 중심**(ETA 대신 “다음 업데이트 시각”), **과장·단정 금지**
* **책임 있는 표현**: “원인 조사 중/완화 적용/복구 검증 중” 단계 구분
* **법적 고지**: 개인정보/보안 사건은 **법적 통지 요건**(지역별)·**수사 협조 안내** 포함

### C. 상태/장애 템플릿(발췌)

**초기 공지(ko)**

```
[KR] Incident: {service} 지연 증가 (감지 시간: {ts})
현재 {region} 리전에서 {service}의 지연이 증가하고 있습니다.
{impact}를 확인하였으며, 완화 조치를 적용했습니다.
다음 업데이트는 {next_update_at}에 게시하겠습니다.
지표: p95 {p95} / 오류율 {err}% (대시보드: {signed_url})
```

**복구 공지(ja)**

```
[JP] 復旧: {service} の遅延は正常化しました ({ts})
影響: {impact}
対処: {action}
モニタリング指標: p95 {p95}, エラー率 {err}%
本件のRCAは{next_update_at}までに共有いたします。
```

**완료/RCA 예고(en)**

```
[EN] Resolved: {service} latency stabilized (at {ts})
Impact: {impact}. Mitigations applied: {action}.
We will publish an RCA by {next_update_at}. Thank you for your patience.
```

### D. 보안 경고/PSIRT(발췌)

```
[SECURITY NOTICE] Suspicious activity detected for tenant {tenant}.
We applied immediate containment (scope: {scope}). Evidence package: {signed_url}.
If you notice anomalies, please contact {contact}. (Ref: PSIRT-{id})
```

* **법적 고지**: 지역별 통지 문구/기한(예: 72h) 템플릿 포함

### E. 고객지원/회신 템플릿(발췌)

* **접수 확인(ko)**: “요청 주셔서 감사합니다. 티켓 번호 {id}로 접수되었으며, **영업일 기준** {sla} 내 1차 회신드리겠습니다.”
* **가이드 링크(en)**: “Please follow the guide below: {signed_url}. If the issue persists, reply to this thread.”
* **일정 조율(ja)**: “面談の候補日時を{window}でご提示ください。オンライン会議リンクをお送りします。”

### F. 자동화/콘솔 연동

* 상태페이지/티켓/메일 템플릿에 **공통 변수** 자동 주입(Region/타임존 변환)
* 템플릿 선택 시 **사전 미리보기**·**i18n 동시 편집**·**감사 로그**(편집/발행) 자동 고정
* **Saved Replies**와 **Notifier**(Day 13/68) 연동

### G. 품질·SLO

.  

* **초기 공지 TAT ≤ 15분**, **업데이트 간격 준수율 ≥ 95%**
* **오역/톤 불일치 0건**(리뷰 2인), **링크 무효 0건**
* 템플릿 사용률·CSAT·재사용률 지표로 지속 개선

### H. API/CLI

* `GET  /v1/templates?type=status|incident|security|support&lang=ko`
* `POST /v1/templates/render { key, lang, vars{} }`
* `POST /v1/templates/publish { key, channels[] }`
* CLI: `plura tmpl list|edit|render|publish`

1) 10:04 KST 탐지 → 10:09 초기 공지(ko/ja/en 동시), 다음 업데이트 10:30 예고.

   → 결과: **표준 문구·i18n 템플릿**으로 커뮤니케이션이 **일관·신속**, 링크/지표/법적 고지까지 **정확**하게 전달.
