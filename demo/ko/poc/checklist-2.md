# 📘 PLURA-XDR PoC 계획서 (Proof of Concept Plan)

## 1. 개요

본 문서는 고객사 환경에서 **PLURA-XDR** 플랫폼의 기능을 검증하기 위한
**PoC(Proof of Concept) 계획서**입니다.
웹·시스템 로그 수집, 에이전트 기반/에이전트리스 기반 탐지,
그리고 실제 공격 시나리오 기반의 탐지·대응 능력을 확인하는 것을 목표로 합니다.

---

## 2. 사전 확인 사항

### 🔍 2-1. 네트워크 및 장비 확인

1. **웹방화벽(모니터랩) 포트 미러링 지원 여부 확인**

   * SPAN / Port Mirroring 기능 존재 여부
   * 미지원 시 별도 TAP 또는 스위치 SPAN 구성 필요

2. **미러링 포트 매체 유형 확인 (광 or UTP)**

   * PoC 장비 인터페이스와 물리 포트 호환성 검토
   * 필요 시 SFP 모듈 또는 UTP 포트 사용 여부 협의

3. **웹방화벽 ↔ PLURA-XDR PoC 장비 간 거리 확인**

   * 동일 랙/층 여부
   * PoE·광케이블·UTP 케이블 길이 확인
   * 배선 경로 및 PoC 장비 설치 위치 확정

---

## 3. PoC 구성 대상

### 🖥️ 3-1. 에이전트 설치 대상

#### Windows

* Windows 11 PC(공격자 역할)
* Windows Server 1대(피공격자 역할)
* 필요 시 추가 서버·PC 포함 가능

#### Linux

* Linux 웹 서버
* Linux 업무 서버(필요 시)

**설치 항목:**

* PLURA-XDR Windows Agent
* PLURA-XDR Linux Agent

---

### 📡 3-2. 에이전트리스 수집 대상(Syslog)

보안 장비·네트워크 장비에서 Syslog 형태로 로그를 직접 수집합니다.

* 시큐아이 (Firewall/IPS)
* 안랩 (Firewall/IPS/보안 게이트웨이)
* IPS 장비 (타사 포함)
* WAF (모니터랩 포함)
* 기타 Syslog 지원 장비

**확인 사항:**

* Syslog 전송 포트(UDP/TCP)
* 로그 포맷(기본/CEF/JSON 등)
* 전송 주체 장비의 정책 및 포트 오픈 필요 여부

---

## 4. 공격 시나리오 (PoC Test Scenario)

### 🚨 4-1. Windows 공격 시나리오

**준비된 장비**

* Windows 11 PC (공격자)
* Windows Server 1대 (피공격 대상)

**공격 플로우**

1. 공격자가 Windows 11 PC에서
   **Windows 서버로 원격 접속 (RDP 등)**
2. 서버 내부에서 권한 상승 시도
3. 인증 정보 수집 및 계정 탈취
4. 동일 네트워크 내 **측면 이동(Lateral Movement)**
5. 후속 공격(서비스 조작·파일 실행 등)

**PoC 확인 포인트**

* 원격 접속 이벤트 탐지
* 계정/권한 관련 로그 분석
* 비정상 프로세스 실행 탐지
* Lateral Movement 흐름 식별
* 의심 네트워크 연결 모니터링

---

### 🐧 4-2. Linux 웹 서버 공격 시나리오

**대상 서버**

* Linux 기반 웹서버

**공격 플로우**

1. 웹 취약점 또는 파일 업로드 기능 악용
2. **웹쉘(WebShell) 업로드**
3. 웹쉘을 이용한 내부 명령 실행
4. 공격자에게 **리버스 쉘(Reverse Connection) 연결**
5. 내부망 스캔 및 추가 공격 시도

**PoC 확인 포인트**

* 웹 로그(WAF/웹서버)에서 웹쉘 업로드 탐지
* Linux OS에서 비정상 프로세스 탐지
* 리버스 쉘 연결 이벤트 분석
* 연결 이후의 내부망 공격 흐름 추적

---

## 5. PLURA-XDR PoC 목표

* 웹·시스템·보안장비 로그의 **통합 수집 및 분석 능력** 검증
* Windows·Linux 모두에서의 **행위 기반 위협 탐지** 확인
* 실제 공격 시나리오 기반에서의 **AI 분석 및 자동 대응 능력** 검증
* 웹쉘·측면이동·권한상승 등 고급 공격 탐지 여부 확인
* PoC 환경 구성의 안정성과 운영 난이도 확인
