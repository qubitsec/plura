# [별첨 A] 마진 · 리베이트 세부표 (최종)

> **적용 범위:** 공공/금융 섹터 총판 계약(독점/준독점 공통)
> **정산 기준:** VAT 제외, 분기 단위 산정 · 연 1회 클로징(정산 옵션 협의 가능)
> **지급 형태:** 선매입 시 *프론트 마진*, 실적 달성분은 *백엔드 리베이트*로 후정산

## 1) 제품군별 기본 마진(Front Margin)

| --------------------------- | ----- | -------: | ------------------------ |
| 구독형 소프트웨어(PLURA-XDR, MSS 등) | 재판매   |  **15%** | **연선불** 계약 시 **+2%p** 가산 |
| 하드웨어/어플라이언스(PLURA-WAF 등)    | 재판매   |  **25%** | 프로젝트성 수주 기준              |

### 1-A) 보조 규칙(구축/서비스 · 연장/갱신)

* **연장/갱신(소프트웨어 유지/구독):** **15%**, **멀티-이어(2–3년) 선결제 시 +2%p**

## 2) 연간 매출 구간별 백엔드 리베이트(Backend Rebate)

|   연간 순매출(억 원) | 리베이트(%) | 추가 가산 조건                                  |
| ------------: | ------: | ----------------------------------------- |
|       **≤ 5** |  **2%** | —                                         |
|  **> 5 ~ 10** |  **3%** | —                                         |
| **> 10 ~ 20** |  **5%** | **신규 로고 3개 이상:** +1%p                     |
| **> 20 ~ 40** |  **6%** | **공공/금융 대형 레퍼런스 1건:** +1%p                |
|      **> 40** |  **8%** | **멀티-이어(2년+) 선결제 30% 이상:** +1%p *(상한 9%)* |

> 리베이트는 **분기 누적 달성률** 기준 분기 말 잠정계산, **연말 클로징** 시 확정.

## 5) MDF(마케팅 공동기금)

* **한도:** 분기 실적의 **3.0%** 내에서 **사전승인** 후 집행
* **용도:** 세미나/부스, PoC 비용 일부, 레퍼런스 콘텐츠 제작
* **정산:** **증빙 기반 실비 정산**, 분기 종료 후 **30일 내**

## 9) “한 장 요약”

* **Front:** SW **15%** / HW **25%** / *(서비스 **10%**, 갱신 **15%**, 연선불/멀티-이어 **+2%p**)*
* **Backend:** **2–8%**(구간) + 성장 **0–2%p** + 전략 가산 *(캡 40%p)*
* **Deal-Reg:** **+3%p**(최초연도, 승인건)
* **MDF:** **3.0%**(분기 실적, 사전승인·증빙 정산)

## 10.1 클라우드 SaaS 형

> **가정:** 연매출 22억(순매출) = **SW 18억 + 서비스 4억 + HW 0억**
> 신규 로고 4개, Deal-Reg 승인 2건(**전부 SW** 8억), 성장률 25%, 대형 공공 레퍼런스 1건

1. **Front Margin**

* SW: 18억 × **15%** = **2.70억**
* 서비스: 4억 × **10%** = **0.40억**
  → **소계 Front = 3.10억**

2. **Backend Rebate** *(구간 >20~40억 = **6%**)*

* 22억 × **6%** = **1.32억**

3. **성장 보너스** *(≥20% → **+1%p**)*

* 22억 × **1%** = **0.22억**

4. **전략 가산**

* 신규 로고 3개 이상: **+1%p** → 22억 × 1% = **0.22억**
* 대형 공공 레퍼런스 1건: **+1%p** → **0.22억**

5. **Deal-Reg 가산(Front 가산)**

* DR 승인 매출 8억(**전부 SW**) × **3%** = **0.24억**

**총 마진/리베이트 = 3.10 + 1.32 + 0.22 + 0.22 + 0.22 + 0.24 = 5.32억**
**유효 마진율 ≈ 5.32 / 22 = 24.2%**

> *연선불 가산 예시:* **SW 50%(=9억)**를 연선불로 체결 시 **+2%p** → **0.18억** 추가(Front 가산).

## 10.2 하드웨어/어플라이언스 형

> **가정:** 연매출 22억(순매출) = **HW 14억 + SW 6억 + 서비스 2억**
> 신규 로고 4개, Deal-Reg 승인 2건(승인 매출 8억 중 **HW 6억 + SW 2억**), 성장률 25%, 대형 공공 레퍼런스 1건

1. **Front Margin**

* HW: 14억 × **25%** = **3.50억**
* SW: 6억 × **15%** = **0.90억**
* 서비스: 2억 × **10%** = **0.20억**
  → **소계 Front = 4.60억**

2. **Backend Rebate** *(구간 >20~40억 = **6%**)*

* 22억 × **6%** = **1.32억**

3. **성장 보너스** *(≥20% → **+1%p**)*

* 22억 × **1%** = **0.22억**

4. **전략 가산**

* 신규 로고 3개 이상: **+1%p** → 22억 × 1% = **0.22억**
* 대형 공공 레퍼런스 1건: **+1%p** → **0.22억**

5. **Deal-Reg 가산(Front 가산)**

* HW 6억 × **3%** = **0.18억**
* SW 2억 × **3%** = **0.06억**
  → **소계 Deal-Reg = 0.24억**

**총 마진/리베이트 = 4.60 + 1.32 + 0.22 + 0.22 + 0.22 + 0.24 = 6.82억**
**유효 마진율 ≈ 6.82 / 22 = 31.0%**

## 11) 용어 정의(약식)

* **Front Margin:** 납품 단가에서 즉시 반영되는 유통 마진
* **Backend Rebate:** 기간 실적 달성률 기반 후정산 인센티브
* **Deal Registration:** 기회 선점/기여를 공식 등록·승인하는 제도
* **MDF:** 공동 마케팅 목적의 집행·정산형 기금
* **신규 로고:** 최초 유료 계약 체결된 최종 고객(자회사/계열사 별도 카운트는 사전 합의)
