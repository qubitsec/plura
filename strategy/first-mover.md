## 1. 제품 경쟁력 강화 – AI 보안 에이전트 고도화 및 XDR 통합 플랫폼

PLURA-XDR은 보안 운영 자동화와 위협 대응의 실행력을 높이기 위해 **AI 기반 SecOps 에이전트**를 중심으로 기능 고도화를 추진하고 있습니다.
웹·API 로그 중심의 탐지 역량을 기반으로 다음과 같은 **3대 에이전트**와 함께 \*\*통합 보안 플랫폼(XDR)\*\*으로 진화하고 있습니다.

---

### ✅ 1-1. AI 기반 SecOps 에이전트 Suite

PLURA-XDR의 핵심 구성 요소는 아래 3종 AI 에이전트입니다.

#### 1) **SOC Copilot – 보안 운영 자동화 도우미**

* 웹로그, 시스템 이벤트 등에서 수집된 다양한 데이터를 분석하여 이상 행위를 식별합니다.
* 탐지 결과에 따라 **자동 티켓 생성, 대응 가이드 제시**까지 지원합니다.
* ChatGPT API 또는 자체 모델을 활용한 **자연어 기반 질의응답(NLQ)** 기능 제공
  → 예: “이 사용자 이상행위 있었어?” 와 같은 단순한 질문도 처리 가능

#### 2) **Auto-Response – 자동 대응 에이전트**

* 위협 탐지 시, **위험도 등급별 자동 대응 조치**를 수행합니다.
  (방화벽 차단, 계정 잠금, 프로세스 종료, 관리자 통보 등)
* 사용자 정의 플레이북 기반의 시나리오 대응 정책으로 WAF, EDR 등과 연동
* **Canary Mode** 및 **빈도 기반 학습** 기능을 통해 **오탐률을 사실상 제로화**
  → *Canary Mode란?*: 정상 행위 기준선을 학습해 벗어날 경우만 탐지하는 방식

#### 3) **Forensic Copilot – 증적 자동 수집 및 리포팅**

* 탐지된 이상 이벤트에 대해 **로그, 프로세스, 파일** 등의 포렌식 데이터를 자동 수집
* 주요 증적에 대해 **블록체인 기반 타임스탬프**를 적용해 공공·법적 감사 대응도 가능

### ✅ 1-2. XDR 통합 플랫폼 포지셔닝

PLURA-XDR은 단순 탐지를 넘어 **대응 및 증적까지 아우르는 All-in-One 보안 플랫폼**입니다.

#### 🔸 All-in-One SECaaS

* WAF, EDR, SIEM, Forensic, Auto-Response 기능을 **단일 콘솔에서 통합 제공**
* 여러 보안 제품을 하나로 통합해 **복잡한 보안 운영 환경을 단순화**

#### 🔸 모듈형 라이트 패키지 제공

* WAF Only, Forensic Only 등 **모듈별 단독 판매가 가능**
* **대기업, 채널 파트너** 대상 **유연한 SKU 구성** 가능 → MSSP, 리셀러 확대 기반

#### 🔸 하이브리드 환경 완전 대응

* **클라우드, 온프레미스, 폐쇄망** 모두 지원
* **오프라인 패치 및 USB 서명체계 내장** → 공공기관 및 방산 시장의 보안 규제 충족

## 🤝 3. 채널 및 파트너 확대 – 퍼블릭 클라우드 MSP 중심 전략

PLURA-XDR은 **보안 운영 자동화 플랫폼**으로서의 경쟁력을 기반으로,
퍼블릭 클라우드 MSP와의 파트너십을 통해 **SECaaS(Security-as-a-Service)** 모델로 전환하고,
중소·중견 기업을 포함한 **대규모 확산형 채널 전략**을 추진합니다.

---

### ✅ 3-1. 전략 개요 – “Managed SECaaS” 전환

* AWS, Azure, GCP 등 **퍼블릭 클라우드 기반 고객사**의 보안 부담과 자동 대응 니즈 증가에 대응하여,
* PLURA-XDR을 \*\*MSP의 보안 상품(MDR, SOC 등)\*\*에 내장시켜 **다중 고객을 대상으로 한 SECaaS 형태**로 제공

#### 🔧 적용 구조

* MSP의 보안 포털 내에 PLURA-XDR의 **분석 및 대응 엔진 내장**
* **Agent 또는 API 방식**으로 신속 도입 가능 (무중단 적용)
* **N-Tier 과금 API**를 통해 고객별 **로그 사용량 및 탐지량 기준 과금**
* **멀티테넌시 SaaS 구조**로 MSP가 다수 고객을 통합 관리

> ☁️ PLURA-XDR은 MSP 보안 포털에 **“탐지 → 대응 → 리포트” 기능을 내장한 보안 엔진**으로 작동하며,
> **MSP의 브랜드로 리패키징하거나 공동 브랜드 상품 구성**도 유연하게 지원합니다.
